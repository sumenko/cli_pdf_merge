# PDF merger tool

Sometimes you have path with many .PDF files

```
.\
MyDir\
	1 - file.pdf
	2 - file.pdf
	... file.pdf
	project.txt
	changelog.txt
```

You run `PDF_MyDir.py` tool and get single file

```
.\YYYY-MM-DD project_name prefix (HHMM).pdf
```

# Install

Just run PDF_merge.py
